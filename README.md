# Industrial Mathematics Projects

Two applied-modelling projects addressing pollutant dispersion and
agent-based urban housing dynamics.

## Overview

This repository contains two group projects completed for the Industrial
Mathematics course at the University of Edinburgh.

### Project 1: Air-Pollution Modelling

The first project models pollutant concentration at different distances from a
waste incinerator and investigates when releases should occur to minimise their
effect on nearby communities.

My contribution was modelling daily and annual wind variation. I developed a
sinusoidal representation using harmonics for annual, daily, and half-daily
cycles.

### Project 2: House Prices and Segregation

The second project investigates how demographics, preferences, and neighbourhood
definitions affect house prices and patterns of residential segregation. The
model uses a Schelling-style agent framework with a utility function that
incorporates similarity across categorical variables.

My contribution was building the baseline model and helping to develop the more
complex model variants.

## Methodology

- Differential-equation and dispersion modelling
- Harmonic modelling of periodic environmental variation
- Agent-based simulation
- Schelling-style residential dynamics
- Utility functions and categorical similarity measures
- Sensitivity and scenario analysis
