# Industrial_Mathematics_Projects
Two group projects completed as part of the "Industrial Mathematics" course at the University of Edinburgh.

Project 1 models the concentration of pollutant at a given distance
away from a waste incinerator. It explore the optimal method and
time for releasing pollutant such that it has minimal effect on the community. My contribution to this project was the modelling of the daily and annual wind variations. I developed a sinusoidal model incorporating three harmonics for annual, daily, and half-daily cycles.

Project 2 aims to understand why house prices differ across a city and investigates the effects of demographics, personal preferences, and neighbourhood radius on house prices and segregation patterns. We investigate it using a Schelling-like model, with a utility update that includes a similarity measure for categorical variables. My contribution to this was to build the baseline model and help developing the more complex models from this. 

The code files for both projects are too large to upload to GitHub but can be shared on request.
